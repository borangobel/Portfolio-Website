Supply Chain Analysis USA

This project analyzes supply chain data from the AdventureWorks2019 database to find useful insights. The analysis looks at important areas like supplier performance, inventory management, demand forecasting, and procurement.

Main Focus Areas:

  • Supplier Performance: Checking how well suppliers are doing to improve purchasing strategies.
  • Inventory Management: Studying stock levels and turnover to better manage inventory and cut down on costs.
  • Demand Forecasting: Predicting future demand to help with planning and resource use.
  • Procurement Analysis: Reviewing purchasing patterns to make better decisions and save money.

    • Dataset: The AdventureWorks2019 database provides detailed information about purchase orders, vendors, products, and sales transactions, which is used for this analysis.

    Tools and Technologies

    • SQL: Used to query and pull data from the AdventureWorks2019 database.
    • Python: Used for cleaning and organizing data, with the help of libraries like pandas.
    • Tableau: Used to create visualizations and interactive dashboards that show the results of the analysis.
