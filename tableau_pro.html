Tableau Project - Data Visualization

This project showcases a dynamic and interactive dashboard created using Tableau, designed to analyze British Airways customer reviews. The dashboard provides a comprehensive overview by allowing users to explore various metrics such as traveler type, seating class, and date range.

Key features include:

  • Global Map Visualization: Displays average customer ratings by country, offering insights into regional satisfaction.
  • Trend Analysis: Charts showing rating trends over time, helping to identify patterns in customer feedback.
  • Aircraft Performance: A detailed breakdown of ratings by aircraft type, highlighting strengths and areas for improvement.

    • This dashboard is a powerful tool for understanding customer satisfaction and guiding data-driven decisions in service delivery.
